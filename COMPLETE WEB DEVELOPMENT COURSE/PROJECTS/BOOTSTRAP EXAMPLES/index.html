Toggleable Tabs and Pills

This is a way to put a lot of content on one page without having to click on a link to a new page. The process is a bit more code intensive but leads to a more dynamic page. Here are the basic steps:

  1. Add data-toggle="tab" to every <a> tag in your list.
  2. Put all your hidden panes in a <div> of class tab-content.
  3. Create a separate <div> for each toggleable pane with class tab-pane.
  4. Add an id to each of your toggleable tab divs and make a link to each of them in your ablove list.
  5. Add and active state to your first list item and first tab pane div.
  6. OPTIONAL: Add a fade in class to each of your tab panes.

Forms -make your website interactive

The functions of forms and inputs are somwhat intertwined, at least that's the way I am understanding them as I am learning. To my understanding, forms are basically groups of inputs, so this section will be about how to create a layout for groups of inputs using Bootstrap.

Helpful tips for forms

  • To make inputs span their parent element by 100%:

    --Add class="form-control" to any <input>, <textarea>, or <select> tags. (A.K.A. textual tags.

    --Put each <label> and <input> inside a separate <div class="form-group">. Doing so will place your label above each text area

  • Vertical Forms:

    Vertical Forms are the default setting for Bootstrap forms, so all you need to do is put everything inside a <form> tag, and of course, follow the tips above. This will arrange your form elements such that the label will be placed above all of your inputs.

  • Horizontal Forms:

    In a horizontal form, labels are aligned right along the edge of each input. If the screen size is decreased, the labels automatically move above each input. It is also necessary to use a bootstrap grid to align your form elements. Here are some steps for achieving this:

    1. Give your <form> element a class of form-horizontal.
    2. Give each <label> element a class of control-label.
    3. As with other types of forms, make sure to put each <label> and <input> inside a separate <div class="form-group"> and add class="form-control" to any <input>, <textarea>, or <select> tags.

  • Inline Forms:

    Inline forms take up one line of a full screen. This is most useful for an unobtrusive login form. You only need to give your <form> element a class of form-inline.

  • Screen Readability:

    It is very important to ensure that your form can be read by a screen reader. That means having labels for every input. If you want to create a form that has placeholders for labels, you can give your labels a class of sr-only so the labels will be visible only to screen readers.

Carousel

Necessary Components

  1. Code that calls on the carousel function
  2. Code that brings up the media and captions
  3. Code that brings up controls

Step 1: Wrap everything in a <div id="(You Decide)" class="carousel slide" data-ride="carousel"> tag.

Step 2: The Carousel Funtion

  1. Put your list items in a <ol class="carousel-indicators"> tag.
  2. For each slide of your carousel, use the following code: <li data-target="#(You decide)" data-slide-to="0" class="active">
  3. For all subsequent slides, eliminate the <class="active" attribute and increase the data-slide-to by one.

Step 3: Adding Media and Captions

  1. Put all your media in a <div class="carousel-inner"> tag.
  2. Put each media item in a <div class="item"> and give your first item a class of active.
  3. Add your <img src and give it style="width:100%".
  4. Put your captions in a <div class="carousel-caption"> tag. Then you can put any size header or caption in, and it will be overlayed on the image.

Step 4: Add controls

  1. For the left control use an <a class="left carousel-control" href="#(YouDecide)" data-slide="prev"> tag.
  2. Inside the <a> tag, add two spans:
    1. <span class="glyphicon glyphicon-chevron-left"></span>
    2. <span class="sr-only">Previous</span>

Modals are popup windows within a page. Clicking anywhere outside the windowor clicking any of the "close buttons will close the modal, so they are very intuitive to use even for people not accostomed to surfing the internet. From what I have learned, modals have two main components: a trigger to open the modal and the content of the modal.

Trigger

  • To a <button> or <a> tag, add the following attributes:
    1. type="button"
    2. class="btn btn-(state) btn-(size)"
    3. data-toggle="modal"
    4. data-target="(You decide)"

Modal Content

Add the following divs:
  1. There are three outer tags
    1. <div id="(You Decide)" class="modal fade" role="dialog">
    2. <div class="modal-dialog"> to which can be added the classes modal-lg or modal-sm to resize the popup box.
    3. <div class="modal-content">
  2. Inside these tags you will put:
    1. <div class="modal-header">
      1. A close button using
        <button type="button class="close" data-dismiss="modal">& times;</button
      2. An optional header using <h class="modal-title">
    2. The body using <div class="modal-body">
    3. The footer using <div class="modal-footer">
      1. A close button using <button type="button" class="btn-btn-default" data dismiss="modal">Close</button>

Tooltips

Tooltips need jquery to work properly. They are basically just tips that appear when you hover ofver the text that is designated with the tooltip data toggle.

How to code a tooltip

A tooltip needs to be placed in an <a> tag. In addition, you have to add some Javascript in order to enable the tooltip stylings for the page.

HTML

<a href="javascript://" data-toggle="tooltip" title="(Text that will appear in the tooltip)"> + LINKED TEXT + </a>

Javascript

<script>
$(document).ready(funtion() {
$('[data-toggle="tooltip"]').tooltip();
});
</script>

Example: Hover over me!

Positioning

If you want to determine the direction in which your popup will appear, add the following attribute to your <a> tag: data-placement= + "top" , "bottom" , "left" , "right" , or "auto" .

Popover

Popovers are similar to tooltips in that they can appear when you click or hover over a designated link but there are a couple important differences. One, they are bigger and have a white background. Two, they can have a header and body text. And three, by default they can only be dismissed and activated by clicking on the designated link (though this attribute can be changed).

How to code a popover

The coding is nearly identical to how a tooltip is coded, but with three changes:

  1. Change tooltip to popover in your <a> tag.
  2. Add data-content="Popover body text goes here" in your <a> tag attributes.
  3. Change tooltip to popover in your Javascript. (NOTE: There are two instances of this to change.)

Closing the popover

By default, the popover can only be closed by clicking on the link again. However, there are two additional options:

  1. Dismiss by clicking outside the popover:
    Do this by adding data-trigger="focus" to your <a> tag.
  2. Open and dismiss by hovering:
    Do this by adding data-trigger="hover" to your <a> tag.

Scrollspy

A Scrollspy is used to highlight the currently the link or button for the visible section of a page> You can see an example of this by opening the dropdown menu on the right and scrolling through the page. Can you see how the different sections are highlighted?

How to code a scrollspy

There are three attributes that you will need to add to your <body> tag.

  1. data-spy="scroll"
  2. data-target=".navbar": you can declare any element as your target, but the most common is the Navbar.
  3. data-offset="50px": this attribute synchronizes the distance from of the screeen and the highlighting of the link. If you have a navbar, then the start of any of your divistions will be around 50px from the top. If you have a vertical pills setup, then you can decrease the distance (maybe to 20px).

You also need to have your page setup with ids for each of your divs so that the navigation works.

Affix

The Affix attribute will make an element stick to the top or bottom of the page while you are scrolling. It can be applied to a navbar or a sidebar. It can also be used in combination with scrollspy.

How to code an Affix

Add data-spy="affix" data-offset-top="(Pixel distance from the top before affix)" to your Navbar or sidebar attributes.

You will also need to add some CSS coding to make everything look just right.
